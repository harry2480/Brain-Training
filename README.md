# 🧠 まいにち脳活スイッチ (Daily Brain Switch)

「まいにち脳活スイッチ」は、1日5分の習慣で頭をシャキッと「オン」にする、シニア向けに最適化された認知機能トレーニング・アプリです。
計算・記憶・判断など、脳の異なる領域を刺激する5つのミニゲームを、インストール不要でブラウザからすぐに楽しめます。
- URL: [https://harry2480.github.io/Brain-Training/](https://harry2480.github.io/Brain-Training/)

## 🌟 コンセプト：1日5分、あたまの切り替え

「トレーニング」という堅苦しいイメージをなくし、朝起きたときや家事の合間に**「脳のスイッチを入れる」**ような軽快な体験を提供します。

### シニアに優しい3つのこだわり

1. **直感的な操作感**: 押し間違いを防ぐ大きなボタンと、一目でわかるアイコンを採用。
2. **即時フィードバック**: 正解・不正解を「まる・ばつ」の大きな視覚演出で伝え、音が出せない環境でも状況がすぐ分かります。
3. **自己ベストへの挑戦**: 過去の自分と競い合うことで、無理なく楽しく継続できる仕組みを導入。

## 🎮 収録ゲーム（スイッチ・メニュー）

| ゲーム名 | 鍛えられる能力 | プレイ内容 |
| --- | --- | --- |
| **計算アタック** | **計算力・集中力** | 4択の中から正しい計算結果を素早く選択します。 |
| **色彩ジャッジ** | **判断力・抑制力** | 文字の色と意味が一致しているかを瞬時に見極めます。 |
| **瞬間メモリー** | **記憶力** | 画面に一瞬だけ現れるパネルの配置を正確に再現します。 |
| **後出しジャンケン** | **瞬発力・柔軟性** | 相手の手に対して「勝ち・負け・あいこ」の指示に素早く反応します。 |
| **数字早押し** | **視野・周辺視野** | ランダムに並んだ1から16までの数字を順番に叩きます。 |

## 🛠 技術仕様 (Technical Stack)

このアプリは、外部サーバーや複雑な設定を必要としない「シングルHTMLファイル」構成です。

* **Framework**: React 18 (CDN経由で軽量に動作)
* **Styling**: Tailwind CSS (モダンで清潔感のあるUIを実現)
* **Data Storage**: ブラウザの `localStorage` を活用し、オフラインでもスコアを保存
* **Icons**: インラインSVG（解像度を落とさず、読み込み速度を最大化）

## 🚀 導入・実行方法

1. `index.html` ファイルを任意の場所に保存します。
2. ファイルをダブルクリックしてブラウザで開くだけで起動します。
* *インターネット環境があれば、最新のライブラリが自動読み込みされます。*

## 📝 難易度調整のヒント

ソースコード内の以下の数値を変更することで、プレイヤーに合わせた調整が可能です。

* **制限時間の変更**: 各ゲームコンポーネント内の `setTimeLeft(30)` の数値を変更。
* **計算の難易度**: `generateProblem` 関数内の `Math.random() * 50` などの範囲を調整。
* **色の種類**: `useColorGame` 内の `colors` 配列に新しい色を追加。
